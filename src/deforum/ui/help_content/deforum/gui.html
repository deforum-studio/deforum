Deforum GUI Overview

Welcome to the Deforum GUI! This page will guide you through the various components and features available in the user interface.

Application Layout

The Deforum GUI consists of several dockable panels, a preview area, and various control options. You can reattach the dockable panels to the tabbed widget or float them as needed. Hereâ€™s an overview of the main components:

Top Menu

The top menu includes several options:

  • File: Basic file operations.
  • Presets: Load and save presets for different settings.
  • Layouts: Manage window layouts.

Render Controls

Located at the top left:

  • Render: Start the rendering process.
  • Stop Render: Stop the rendering process.
  • Tile Subwindows: Organize subwindows in a tiled layout.
  • Save Preset: Save the current settings as a preset.
  • Add to Batch: Add the current render to the batch queue.
  • Load Render: Load a previously saved render.
  • Disable Controls: Disable all interactive controls.

Preview Area

The preview area shows the current state of your animation:

  • Video Player: Preview the animation video.
  • Image Preview: Preview the generated images.

Control Panels

The control panels allow you to manipulate various parameters in real-time. These panels are dockable and can be floated as needed. Here are the key panels:

  • Controls: Audio path, milk path, and generate VFX options.
  • Diffusion Schedules: Adjust schedules for steps, strength, noise, and CFG scale.
  • Animation Motion: Control parameters like angle, zoom, translation, and rotation.
  • Live Controls: Joystick and keyboard controls for movements.

Autosave and Presets

The application autosaves the last settings and window state. You can load presets from the top menu under the "Presets" option.

Using the GUI

To get started with the Deforum GUI:

  1. Load a preset or adjust the settings manually using the control panels.
  2. Use the render controls at the top left to start and stop rendering.
  3. Preview your animation in the video player and image preview areas.
  4. Use the live controls for real-time manipulation of parameters.
  5. Save your settings as a preset for future use.

Tips

  • You can drag and dock panels to customize your workspace.
  • Use the "Tile Subwindows" option to organize floating panels.
  • Regularly save your presets to avoid losing settings.

We hope this guide helps you navigate and utilize the Deforum GUI effectively. Happy animating!
